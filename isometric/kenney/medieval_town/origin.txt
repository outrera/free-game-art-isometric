This pack includes walls, roofs, floors and roads to create a medieval town. All pieces are modular and allow you to create thousands of different structures.

3D Preview (Sketchfab)

Features:

    Works in Unity, Unreal and all other 3D engines
    Optimized, low poly models
    Modular in design
    No textures (just materials)
    Easy to model additional elements
    Unity package included
    Properly named

Files:

    Separate OBJ + MTL files (67x)
    Unity package (1x)
