This package includes prototype isometric tiles, walls and objects. Included are various blocks, primitives, switches, doors and stairs. You can purchase a bundle including many more isometric tiles like there here.

Files:

    Separate PNG files (180x)
    Tiled template
    Bonus: Top-down view rendering of each tile!

Technical:

    Rendered at 35.264° × 40° viewpoint
    Image (tile) size is 256 × 512 pixels
    Tile width is 256 pixels, height varies
    PNG images with alpha channel

Let me know if you have any questions or requests!
